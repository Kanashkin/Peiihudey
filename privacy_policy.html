Политика конфиденциальности

Приложение «Ешь и худей»

Последнее обновление: 26 июня 2026 г.

Настоящая Политика конфиденциальности описывает, как приложение «Ешь и худей» обрабатывает данные пользователей.

1. Сбор данных

Приложение «Ешь и худей» не собирает, не передаёт и не хранит персональные данные на внешних серверах. Все данные хранятся исключительно на устройстве пользователя.

2. Данные Apple Health (HealthKit)

Приложение запрашивает доступ к Apple Health для чтения следующих данных:

  • Количество шагов
  • Активные калории

Эти данные используются исключительно для отображения внутри приложения. Данные HealthKit никогда не передаются третьим лицам, не загружаются на серверы и не используются в рекламных целях.

3. Локальное хранение данных

Приложение хранит следующие данные локально на вашем устройстве:

  • Параметры профиля (пол, возраст, рост, вес, уровень активности)
  • Дневной журнал еды и активности
  • История дней
  • История веса

Все данные хранятся локально и могут быть удалены путём удаления приложения.

4. ИИ-сканер еды (Premium)

При использовании функции ИИ-сканера фотография еды отправляется на защищённый сервер для анализа с помощью искусственного интеллекта. Изображение используется только для определения калорийности и не сохраняется на сервере после обработки запроса.

5. Подписка Premium

Для оформления подписки используется сервис RevenueCat. Платёжные данные обрабатываются Apple через App Store и не передаются разработчику приложения.

6. Уведомления

Приложение может отправлять локальные push-уведомления в качестве напоминаний. Уведомления формируются на устройстве и не требуют передачи данных на серверы.

7. Сторонние сервисы

Приложение не интегрировано с рекламными сетями и не передаёт данные третьим лицам, за исключением случаев, описанных в разделе об ИИ-сканере.

8. Возрастные ограничения

Приложение предназначено для пользователей старше 18 лет и не предназначено для сбора данных детей.

9. Изменения политики

Мы оставляем за собой право обновлять настоящую Политику конфиденциальности. Об изменениях будет сообщено через обновление приложения.
